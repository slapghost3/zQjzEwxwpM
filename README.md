# Python仓库管理系统 python472

## 项目概述

本项目是基于Python开发的仓库管理系统，采用 Django 框架进行后端开发，Vue.js 进行前端开发，使用 MySQL 作为数据库管理系统。该系统实现了商品管理、分类管理、用户管理、日志管理以及系统信息展示等基础功能。

## 技术栈

- 后端：Python 3.x，Django 3.x
- 前端：Vue.js，Element UI
- 数据库：MySQL
- 运行环境：推荐使用 Linux 或 macOS，Windows 环境亦可

## 功能模块

- **商品管理**：支持商品的增删改查操作
- **分类管理**：实现对商品分类的管理
- **用户管理**：管理系统用户账户
- **日志管理**：记录系统操作日志，便于追踪
- **系统信息**：展示系统相关信息
- **批量删除**：支持批量删除操作
- **分页查询**：提升数据检索效率

## 系统角色

- 系统管理员：负责系统的日常管理与维护
- 普通用户：使用系统的各项功能

## 运行环境

- Python 3.x
- Django 3.x
- Vue.js
- Element UI
- MySQL

## 部署步骤

1. 配置 Python 环境及 Django
2. 安装前端依赖库
3. 配置 MySQL 数据库
4. 运行后端服务
5. 运行前端服务
